# Agent Skills Reference — Creative & Design

This folder contains detailed skill prompts you can give to any agent. Each file is a self-contained guide that teaches the agent how to perform that skill at production quality.

## Files

| File | Skill | Purpose |
|------|-------|---------|
| `01-FRONTEND-DESIGN.md` | Frontend Design | Build distinctive, production-grade UIs |
| `02-ANIMATE.md` | Animate | Add purposeful animations and micro-interactions |
| `03-COLORIZE.md` | Colorize | Strategic color introduction |
| `04-TYPESET.md` | Typeset | Typography refinement |
| `05-BOLDER.md` | Bolder | Amplify safe/boring designs |
| `06-DELIGHT.md` | Delight | Add moments of joy and personality |
| `07-POLISH.md` | Polish | Final quality pass before shipping |
| `08-CRITIQUE.md` | Critique | Evaluate design effectiveness |
| `09-HUMANIZER.md` | Humanizer | Remove AI writing patterns |
| `10-DIAGRAM-MAKER.md` | Diagram Maker | Create SVG/Excalidraw diagrams |
| `11-MEME-MAKER.md` | Meme Maker | Generate memes from templates |
| `12-CANVAS.md` | Canvas | Present HTML on connected devices |

## How to Use

1. Give the agent the relevant file(s) as a system prompt or context
2. Describe what you want built
3. The agent will follow the skill's methodology

## Recommended Order

For a new project:
1. Frontend Design (establish direction)
2. Colorize + Typeset (foundation)
3. Animate + Delight (life and personality)
4. Polish (final pass)
5. Critique (verify quality)

For an existing project:
1. Critique (identify issues)
2. Pick relevant skills to fix
3. Polish (final pass)
