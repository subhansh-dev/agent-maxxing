# Code Generator — Multi-Language Code Scaffolding

> Describe what you need, get working code. Multiple languages, ready to run.

## Key Features

- **Function Generation** — Describe it, get a complete annotated function
- **Class Generation** — OOP design with constructor and methods
- **API Endpoints** — RESTful routes and handlers
- **CRUD Operations** — Full Create/Read/Update/Delete code
- **Test Code** — Auto-generate unit tests with assertions
- **Refactoring** — Optimization suggestions with examples
- **Language Conversion** — Conversion guide between Python/JS/Go/Java
- **Project Boilerplate** — Quick-start project skeleton

## Supported Languages

Python, JavaScript, TypeScript, Go, Java, Rust, PHP, Ruby, C#, Shell

## Usage

Describe what you need in natural language. The agent generates complete, runnable code.

Every output includes:
1. Language label and suggested filename
2. Complete runnable code
3. Inline comments
4. Usage example
